## OpenWiki

This repository uses OpenWiki for recurring code documentation. Start with `openwiki/quickstart.md`, then follow its links.

This repository does not have a scheduled OpenWiki updater. Update the wiki in the same pull request when a change affects architecture, flows, domain, endpoints, or operations.

Ground every claim in source you actually read. When existing docs contradict the code, prefer the code and flag the stale claim.
